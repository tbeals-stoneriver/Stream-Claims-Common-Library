Stream Claim XSD Documentation

Stream Claim Schemas are assigned to the localnamespace and all schema element exetensions are resolved locally within the same namespace. The Common schema element Types are Abstracted for better reuse.The schema follows the same hiererachy pattern used by our java object model and is consistent to avoid any namespace collisions. The Element and Attributes names follow the java bean spec naming convention and they are kept inline for cross mapping and analysis between schema and DTO.



Stream uses XStream - A Lightweight open source marshalling tool to convert between xml and dto and vice versa using an internal alias mapping file to map the Elements of the xml.
